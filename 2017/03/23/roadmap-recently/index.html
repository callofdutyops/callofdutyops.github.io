Roadmap recently

发表于 2017-03-23 | 分类于 Life

最近感觉很多应该掌握的技术虽然都知道，也用过，也能说上一知半解，但是没有成系统串起来，趁着研一下学期正好课不多也没科研任务，好好梳理一下需要重新掌握的知识。

  1. 语言
    • C++：阅读《C++ Primer》，25天
    • Python：重温，着重温习高级特性
    • Java：重温，重拾老本行：）
  2. 算法
    • 基本算法：阅读《算法》，45天
  3. Linux
    • Shell 脚本：阅读《Linux Shell 脚本攻略》，18天
  4. 异构计算
    • CUDA：按照这个Learning path学习基础知识，阅读《CUDA C Programming Guide》以及《CUDA C Best Practices Guide》，60天
    • OpenCL：暂无具体打算，作为了解。
  5. 机器学习
    • 数据挖掘基础理论：阅读《数据挖掘导论》，25天
    • 深度学习理论学习：阅读《Deep Learning》，45天
    • Python爬虫：Scrapy，45天
    • 框架：TensorFlow
  6. 数学理论
    • 矩阵分析：《特殊矩阵》课程，阅读《矩阵分析与应用》，45天
    • 图论：《图论》课程，45天
    • 阅读Deep Learning的paper road，200天
坚持原创技术分享，您的支持将鼓励我继续创作！
